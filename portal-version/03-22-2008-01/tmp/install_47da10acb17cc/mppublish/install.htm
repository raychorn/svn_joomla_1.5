INSTALL

  1. Extract the archive
  2. Install the Component com_mp.zip
  3. Copy the mppublish directory to your joomla install root on your webserver.
  4. Install the MGMEDIABOT you can find it in on http://extenstions.joomla.org site.

Configuration

Create a new menu itemfor the component see image below

  1. Enter all your menu parameters
  2. Select UploadBoxPro From the coomponent list
  3. Click to make available and edit the Component Parameters

Cofigure the component parameters see below. For instructions on each field simple hover your mouse pointer over the field names.

CHANGING THE DEFAULT PARAMETERS.

If you changed the default Joomla db prefix for jos_ you will need to change them in the some of the files located in the mppublish folder my_clipboxtv

NOTE: "my" represents you current db prefix

mpublish/upload.php
mppublish/addarticle.php

edit the following lines

upload.php: jos_clipboxtv >> my_clipboxtv

addarticle.php: jos_categories = my_categories and jos_content = my_content
